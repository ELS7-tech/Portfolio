Sobre Mim

Sou Ederson Lourenço dos Santos, Analista de TI com especialização em desenvolvimento de sistemas e análise de dados. Possuo expertise em linguagens de programação como PHP e Java, além de conhecimento em bancos de dados SQL e ferramentas de visualização de dados como Power BI. Dedico-me à criação de soluções tecnológicas inovadoras e eficientes. Confira alguns dos meus projetos abaixo!

Demonstração de Projetos

Agenda Eletrônica da Saúde

Sistema de agendamento de consultas, integrado entre unidades de Saúde. Feito com PHP, HTML, CSS, JavaScript e SQL.

Site Igreja Assembleia de Deus

Site integrado com sistema de gerenciamento de membros, eventos e finanças.

Sistema Meu Gabinete

Sistema de gerenciamento de Gabinete Parlamentar, para gerenciar agenda, demandas, reuniões, atas e proposições legislativas.

Dashboard BI Visitas

Dashboard em Power BI para acompanhamento de visitas domiciliares e sincronização dos Agentes Comunitários de Saúde, usando o banco de dados e-SUS.

Dashboard BI Unidades

Dashboard em Power BI para gestão e visualização das Unidades de Saúde e geolocalização.

Dashboard BI Imunização

Dashboard em Power BI para acompanhamento de indicadores e registros de imunização, por unidade e por paciente.
